Deep Learning-Based Liver Cancer Segmentation Using U-net and DALU-Net

Used U-net and DALU-Net (Deep Attention LSTM U-Net) to train on the 3D IRCADb dataset for liver tumor segmentation. Achieved a dice coefficient of 0.899 using DALU-Net.

Evaluation of Subarachnoid Space Volume Using Segmentation Techniques in MRI Imaging

Aligned T1w and T2w images to a 1mm MNI space, employed FastSurfer for ventricle segmentation in brain MRI, synthesized T2w images using CycleGAN, and calculated SAS volume in T2-weighted MRI images.

Epoch Extraction for Speaker Differentiation Using Zero Frequency Filtering

Designed a Zero Frequency Filter to extract the Epoch (glottal closure instant) from 1132 utterances collected from CMU_ARCTIC speech synthesis databases. Implemented a GMM for training and compared the accuracy with other filters like ZFR and ZPZFR. The filter was finally implemented on a VLSI board using VHDL.

Machine-Learning-Based Skin Lesion Segmentation and Classification

This project focused on segmenting skin lesions and classifying them into three types: Common Nevi, Atypical Nevi, and Melanoma. The dataset was processed using Otsu's method for lesion segmentation and symmetry detection. The extracted features were used in a machine learning approach to predict lesion type, with a final evaluation based on various metrics.

Paper | GitHub Code

Active Learning for Isolated Sign Language Recognition Using Depth Information

Developed a model to predict depth information from sign language datasets such as WLASL, GISLR, and AUTSL. Extracted body landmarks and applied a transformer-based model combined with active learning techniques to recognize sign language. Achieved 96% accuracy on the tested datasets.
